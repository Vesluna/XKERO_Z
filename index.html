XKERO_Z // UltraBox Companion

A two-part toolkit for UltraBox — the open-source chiptune tracker and sampler.
Manage custom sample URLs and sort your audio libraries with ease.

🎵 UltraBox Companion

Embed the UltraBox editor, build custom sample URL lists, and open them in UltraBox with one click. Persists your URLs in localStorage.

⚙ Sample Compiler

Upload a ZIP of audio files. The compiler classifies them into a structured folder hierarchy by instrument type and packages them for download.

Open UltraBox ↗ View on GitHub ↗

The Companion & Compiler require a local Python/Flask server to run. See the README for setup instructions. UltraBox itself runs entirely in your browser — no server needed.
