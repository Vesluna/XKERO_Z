# XKERO_Z Tools Suite

A professional local-first audio utility suite for producers and sound designers.

## Tools Included

### 1. Sample Compiler
Organize your messy audio libraries into a structured, categorized format.
- **Smart Matching:** Uses whole-word keyword detection for precise categorization.
- **Instrument Taxonomy:** Automatically sorts sounds into 100+ subcategories (Strings, Brass, Drums, Synths, etc.).
- **Nested ZIP Support:** Automatically handles ZIP files found inside your library.
- **SF2 Sorting:** Optionally alphabetizes your SoundFont collection.

## How to Run Locally

1. **Install Python:** Ensure you have Python 3.8+ installed on your system.
2. **Download & Extract:** Download the XKERO_Z zip file and extract it to a folder.
3. **Install Dependencies:**
   Open your terminal/command prompt, navigate to the extracted folder, and run:
   ```bash
   pip install -r requirements.txt
   ```
4. **Launch the Suite:**
   Run the following command:
   ```bash
   python app.py
   ```
5. **Access the Tools:**
   Open your web browser and go to:
   `http://127.0.0.1:5000`
